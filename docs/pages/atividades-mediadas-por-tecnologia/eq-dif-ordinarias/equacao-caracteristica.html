Determinando a Equação Característica

Introdução

A escolha da equação característica é comumente utilizada ao resolver equações diferenciais lineares de segunda ordem com coeficientes constantes. Este método transforma a equação diferencial em uma equação algébrica, que é geralmente mais fácil de resolver. Porém, só é aplicável a tipos específicos de EDOs

A forma geral de uma Equação Diferencial Linear de Segunda Ordem Homogênea com Coeficientes Constantes é:

ay'' + by' + c*y = 0

onde y'' é a segunda derivada de y, y' é a primeira derivada, e a, b, e c são constantes.

A equação característica associada a essa equação diferencial é:

ar^2 + br + c = 0

onde r são as raízes da equação.

A solução geral da equação diferencial será determinada pelas raízes r desta equação característica.

  • Se as raízes r1 e r2 são reais e distintas, a solução geral será y = C1e^(r1x) + C2e^(r2x).
  • Se as raízes são reais e iguais (r1 = r2 = r), a solução geral será y = (C1 + C2*x)e^(rx).
  • Se as raízes são complexas (r = α ± βi), a solução geral será y = e^(αx)(C1cos(βx) + C2sin(βx)).

Nessas soluções, C1 e C2 são constantes que podem ser determinadas por condições iniciais ou condições de contorno.
